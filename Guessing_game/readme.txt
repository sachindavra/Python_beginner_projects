How this works:
This is a guessing game. Where a list of strings/numbers are hard coded. A random string/number will be picked up. In case of alphabet game, few alphabets of it will be printed on the screen while it will show "_" for remaining alphabets. For example: p_ll_w. Correct word is pillow, Which you will have to guess. In case of numbers, you will have to guess a random number and it will show hint as you will guess wrong. 
You will get 10 chances to guess correct answer. As you will be guessing right, It will keep printing write guesses else decrese your changes of guessing by 1.
If you guess all the charactors in 10 chances, you will win else loose the game. 

Instruction:
Simply run the script as normal python script and happy guessing.
