How this works:
This is a guessing game. Where a list of strings will be given. A random string will be picked up and few alphabets of it will be printed on screen while it will show "_" for remaining alphabets. For example: p_ll_w. Correct word is pillow, Which you will have to guess. 
You will get 10 chances to guess correct. As you will be guessing write it will keep printing write guesses else decrese your changes of guessing by 1.
If you guess all the charactors in 10 chances, you wil will else loose the game. 

Instruction:
Simply run the script as normal python script and happy guessing.
